Class: StateMachine

StateMachine

Provides safe transitions between states

Constructor

new StateMachine(canvas)

Parameters:
Name Type Description
canvas canvas canvas on which to draw board
Properties:
Name Type Description
states array Array of all states.
currentState String The current state of the state machine.
constructors array Array of all constructor functions for all states.
transitions array Array of all transition functions for all states.
destructors array Array of all destructor functions for all states.
start(string_state) function Starts the StateMachine in the given state. (Calls contructors for that state)
createState(string_state(s)) function Add state or list of states to the StateMachine. Ex: "state1, state2"
createTransition(string_transition,function_transition) function Adds transition between two states or groups of states and a transition function which will be run during the transition (between the destructor and constructor). Ex: "(state1, state2) -> state3", "state1 <-> state2"
createConstructor(string_state(s),function_constructor) function Adds a constructor for a state or list of states which will run when entering the state.
createDestructor(string_state(s),function_destructor) function Adds a destructor for a state or list of states which will run when leaving the state.
transitionTo(state) function Transitions to the provided state. Calls destructor(s) for currentState, transition function(s) to new state, and constructor(s) for new state.
addParent(stateMachine,state) function Adds a parentState (of another stateMachine, parentMachine) for this StateMachine so that the parentMachine will transtion to the given parentState when a transtionTo is called of this stateMachine.
